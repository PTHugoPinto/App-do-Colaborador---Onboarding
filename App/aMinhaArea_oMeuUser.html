Os direitos administrativos dos postos de trabalho para todas as contas foram removidos. Quando os direitos administrativos forem exigidos, irá aparecer o popup UAC, mas o nome de utilizador e a palavra-passe não permitirão direitos de administração. Em vez disso, os utilizadores terão de solicitar uma conta de administração temporária para a sua estação de trabalho a partir do Portal de Gestão de Conta de Diretório Ativo (ADAM).

Poderei utilizar as minhas credenciais de utilizador para interromper tarefas no Task Manager?

  • Ainda poderá operar o Task Manager, mas apenas para as tarefas que definiu; As tarefas do sistema precisam de acesso administrativo para interagir com o que não será capaz de parar estas tarefas.

Ainda poderei instalar aplicações?

  • Ainda poderá descarregar e instalar aplicações a partir das aplicações/software fujitsu, uma vez que estas são entregues via SCCM. Você NÃO será capaz de descarregar software aleatório de outras fontes.

Ainda poderei usar as minhas smart printer apps sem direitos de administração?

  • Não será possível utilizar Apps como a HP Smart com uma conta padrão.
